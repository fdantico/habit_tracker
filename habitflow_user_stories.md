# HabitFlow User Stories

## Personas

### 1. Marco – Young Professional
Wants to build productive morning and evening routines.
Goal: Stay consistent and track progress over time.

### 2. Sara – University Student
Wants to build study and health habits.
Goal: Improve discipline and reduce procrastination.

## Login & Registration User Stories

- As a new user, I want to register with username and password so that I can create a personal account.

- As a registered user, I want to log in with my username and password so that I can access my habits and progress.

- As a user, I want to receive an error message if I enter incorrect login details so that I understand what went wrong.

- As a user, I want validation messages if I leave fields empty so that I can complete the form correctly.

- As a user, I want my account information to be stored in local storage so that my data persists between sessions.

## Home Screen User Stories

- As a user, I want to see an overview of my daily habits so that I can quickly check my progress.

- As a user, I want to see my current streaks so that I stay motivated.

- As a user, I want quick access buttons to add a new habit so that I can easily update my routine.

- As a user, I want to see completion percentage for today so that I can track daily performance.

## Habit Detail Screen User Stories

- As a user, I want to view detailed statistics for a habit so that I can track long-term consistency.

- As a user, I want to mark a habit as completed for the day so that I can track progress.

- As a user, I want to edit or delete a habit so that I can manage my goals.

- As a user, I want to see related habit suggestions so that I can improve my routine.

## Persistent Data User Stories

- As a user, I want my login session to remain active so that I don’t need to log in every time.

- As a user, I want my habits and progress history to be saved so that I do not lose my data.

- As a user, I want my theme preference (light/dark mode) to persist so that the app remembers my settings.

## External API User Stories

- As a user, I want to see a motivational quote of the day so that I feel encouraged to complete my habits.

- As a user, I want to see weather information so that I can plan outdoor habits accordingly.

- As a user, I want to sync habits with my calendar so that I can manage my schedule efficiently.

## Settings Menu User Stories

- As a user, I want to access the settings menu from any screen so that I can update preferences easily.

- As a user, I want settings categorized (Account, Notifications, Appearance) so that navigation is simple.

- As an admin, I want to enable or disable specific features so that I can maintain system security.

## Settings Screen User Stories

- As a user, I want to enable dark mode so that I reduce eye strain.

- As a user, I want to update password so that I keep my account secure.

- As a user, I want to customize reminder times so that notifications fit my schedule.

## Notifications User Stories

- As a user, I want to receive daily reminders for incomplete habits so that I stay consistent.

- As a user, I want to disable promotional notifications so that I only receive important alerts.

## Prioritization

High Priority:
- Registration & Login
- Add / Complete Habit
- Data Persistence

Medium Priority:
- Habit statistics
- Notifications

Low Priority:
- External API integration
- Advanced analytics
